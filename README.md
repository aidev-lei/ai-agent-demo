# AI Agent Demo - 智能客服机器人

一个基于RAG（检索增强生成）的智能客服系统Demo，可用于展示给客户。

## ✨ 功能特性

- 📚 基于文档的智能问答（RAG）
- 💬 多轮对话支持
- 🔧 工具调用（查询订单、计算价格等）
- 🎯 引用溯源（显示答案来源）
- 📊 对话历史记录

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 OpenAI API Key
```

### 3. 准备知识库

将你的FAQ文档放入 `data/` 目录，支持 .txt, .md, .pdf 格式。

### 4. 启动系统

```bash
# 首次运行需要构建向量库
python src/build_index.py

# 启动对话界面
python src/chat.py
```

## 📁 项目结构

```
ai-agent-demo/

```
